## Design principles (summary)

1. **Dark premium fitness** — never corporate/admin-template aesthetics.  
2. **Lime = body / workouts / progress** · **Purple = AI only** — never swap semantics.  
3. **Bottom nav, 5 tabs** — AI Coach center, elevated, purple gradient.  
4. **Photos & numbers are heroes** — large KPI typography, visual progress timelines.  
5. **Bilingual by design** — English LTR + Persian RTL mirrored at component level.
